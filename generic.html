Graduation

Therefore, my first action was to try to become a member with classmates. Share the knowledge and show good teamwork drag us to success and adapt to new environments. If I start to learn new knowledge, then I will show my passion to perfectly understand that.

Projects

After I successfully graduated the college and got a diploma, begin to build projects. Among them, my favourite project was the Electrical Work Package which customizes the location of heavy electrical equipment in the given spaces. I loved using AutoCAD, so when I working on this project, spend the time on this software and improved the desgin and drawing skill.

This project was my final project from NAIT; Examination of the Protection Application of the Overreach of Distance Relay. This project simulates the main cause of the blackout which is distance relay can not protect the fault outside of the protection. Therefore, this project simulates why this relay can not protect and how to make it available to protect from the fault with ETAP software.

The Arduino Thermometer is a simple project. Using the Arduino controller Board and the DHT11temperature/Humidity Module received and control the data by Arduino software to show up in LCD.

The Lol level search is a new challenge. Therefore, this website is simple as using Jquery and HTML. Using the Riot API, when the user types the name of the League of Legend name, then request and get API by Jquery and represent the level of the name. However, the Riot API is only valid 24hours, so every single time tries to use the website, have to get an AIP key.
